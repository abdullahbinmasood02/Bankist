# Bankist Web Application

## Overview

Bankist is a professional, minimalistic banking web application built to demonstrate modern JavaScript techniques, array methods, and DOM manipulation. The application allows users to securely log in, view and manage their accounts, transfer funds, request loans, and close accounts. It is designed for educational purposes and highlights best practices in JavaScript and user interface design.

## Features

- User Authentication: Secure login using username and PIN.
- Multiple Accounts: Supports multiple user accounts with unique credentials.
- Transaction History: Displays a detailed list of all deposits and withdrawals, including transaction dates and types.
- Balance Calculation: Real-time calculation and display of the current account balance.
- Summary Section: Shows total deposits, withdrawals, and interest earned for each account.
- Money Transfer: Allows users to transfer funds to other registered users within the application.
- Loan Requests: Users can request loans if eligible, based on their deposit history.
- Account Closure: Securely close an account by confirming credentials.
- Logout Timer: Automatic logout after a period of inactivity to enhance security.
- Responsive User Interface: Clean, modern, and responsive UI styled with CSS.
- Date Formatting: Transaction dates are formatted based on user locale and recency (e.g., Today, Yesterday, X days ago).
- Currency Formatting: All monetary values are formatted according to the user's locale and account currency.
- Sorting Transactions: Users can sort their transaction history by amount.

## Technologies Used

# Bankist App

## Overview

Bankist is a minimalistic banking web application designed to demonstrate modern JavaScript array methods and DOM manipulation. The app allows users to log in, view their transaction history, transfer money, request loans, and close accounts. It is intended for educational purposes and showcases best practices in JavaScript and UI design.

## Features

- **User Authentication:** Log in with a username and PIN to access your account.
- **Account Types:** Supports multiple account types (Standard, Basic, Premium).
- **Transaction History:** View a list of all deposits and withdrawals, with sorting functionality.
- **Balance Calculation:** Real-time calculation and display of current account balance.
- **Summary Section:** Displays total deposits, withdrawals, and interest earned.
- **Money Transfer:** Transfer funds to other users within the app.
- **Loan Requests:** Request a loan if eligible (based on deposit history).
- **Account Closure:** Securely close your account by confirming credentials.
- **Logout Timer:** Automatic logout after a period of inactivity.
- **Responsive UI:** Clean and modern interface styled with CSS.

## Technologies Used
